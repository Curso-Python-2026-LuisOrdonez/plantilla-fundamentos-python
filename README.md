# Unidad 1: Fundamentos de Python

Bienvenido a tu primera práctica práctica. El objetivo es aplicar lo aprendido sobre **variables, operadores y sentencias básicas** en Python.

## Instrucciones Generales
1.  Abre esta carpeta en **VS Code**.
2.  Crea un archivo nuevo llamado `main.py` dentro de la carpeta.
3.  Resuelve los ejercicios en ese archivo (puedes usar comentarios `#` para separar cada ejercicio).

---

### Ejercicio 1: Identificación (Variables)
Crea un programa que declare tres variables con tus datos reales:
* `nombre`: (texto) Tu nombre completo.
* `edad`: (entero) Tu edad.
* `estatura`: (float) Tu altura en metros (ej. 1.75).

**Requerimiento:** Usa `print()` para mostrar una frase completa concatenando estas variables.
> *Ejemplo: "Hola, me llamo Luis, tengo 25 años y mido 1.75m."*

---

### Ejercicio 2: Operadores Matemáticos
Declara dos variables numéricas, `numero_1` y `numero_2` (asígnales los valores que quieras).
Realiza e imprime el resultado de las siguientes operaciones:
1.  Suma.
2.  Resta.
3.  Multiplicación.
4.  División.

---

### Ejercicio 3: Sentencias de Control (Decisiones)
Usando la variable `edad` que creaste en el Ejercicio 1, escribe una sentencia **if/else**:
* **Si** la edad es mayor o igual a 18, imprime: *"Es mayor de edad"*.
* **Sino** (else), imprime: *"Es menor de edad"*.

---

## Cómo entregar
1.  Asegúrate de que tu código funcione ejecutándolo en la terminal.
2.  Guarda los cambios (`Ctrl + S`).
3.  Realiza el **Commit** y **Sync Changes** (Push) desde la pestaña de Source Control de VS Code.
